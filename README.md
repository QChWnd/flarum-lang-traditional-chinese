## Flarum 正體中文語言包 / Traditional Chinese Language Pack

### 需求 / Requirement
  - Flarum v1.0 and newer

### 安裝 / Installation
```
composer require qchwnd/flarum-lang-traditional-chinese -v dev-master
```

### 移除 / Remove
```
composer remove qchwnd/flarum-lang-traditional-chinese
```
